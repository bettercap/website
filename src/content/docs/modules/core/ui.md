A module to use bettercap's web UI.

## Commands

### `ui on`

Activates the web UI.

## Parameters

| Parameter    | Default     | Description                        |
| ------------ | ----------- | ---------------------------------- |
| `ui.address` | `127.0.0.1` | Address to bind the web ui to.     |
| `ui.port`    | `8080`      | Port to bind the web ui server to. |
