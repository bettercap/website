A module to manage bettercap's UI updates and installed version.

### Commands

#### `ui.on`

Activates the weui.

### Parameters

| Parameter    | Default     | Description                        |
| ------------ | ----------- | ---------------------------------- |
| `ui.address` | `127.0.0.1` | Address to bind the web ui to.     |
| `ui.port`    | `8080`      | Port to bind the web ui server to. |
